# f02 — Version Control

Companion repository for the **[f02 — Version Control](https://thecodingidiot.com/chapters/f02-version-control)** chapter on [thecodingidiot.com](https://thecodingidiot.com).

---

## Follow my journey

You are working through the implementation pages. Clone this repository alongside your practice repo, then run the tester from inside it.

```bash
# from the same parent directory as your f02-version-control repo
git clone https://github.com/thecodingidiot-com/f02-version-control.git f02-tester

cd f02-version-control
bash ../f02-tester/test.sh
```

The tester checks the shape of your repository — commit count, merge history, conflict resolution, and remote configuration. Run it after completing the implementation pages.

---

## Follow your journey

You built it independently. Run the tester against your completed repository:

```bash
git clone https://github.com/thecodingidiot-com/f02-version-control.git f02-tester

cd f02-version-control
bash ../f02-tester/test.sh
```

The tester prints `PASS` or `FAIL` with a short explanation for each check. Run `bash ../f02-tester/test.sh --help` to see what each check verifies.
